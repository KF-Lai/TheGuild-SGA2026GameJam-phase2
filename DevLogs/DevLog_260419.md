### 全局規範建立

- **資料表 ID 型別原則**：int 用於 PK/FK/可擴展列舉；string 僅用於等級符號（F~SSS）與語義標籤
- **CSV null sentinel 規範**：int 欄位空值填 `0`，不得留白；string 空值填空字串
- **ID 命名規範**：欄位名稱中的「ID」縮寫一律大寫（`professionID`、`typeIDs`，不得寫 `professionId`、`typeIds`）
- **全域 ID 重命名**：背景 agent 將 5 份 GDD 中所有 `Id`/`Ids` 統一更正為 `ID`/`IDs`
