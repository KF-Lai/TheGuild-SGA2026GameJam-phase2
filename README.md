# The Guild
> **狀態**：Game Jam phase 2 版本

---

### 遊戲概念

**The Guild** 是一款**放置型經營遊戲**，玩家做為新上任的公會長，要招募冒險者、接受委託、派遣任務、管理資源，並發展與經營公會。

### 核心循環

```
接受委託（任務出現）
    ↓
招募冒險者（候選池刷新）
    ↓
派遣任務（玩家推薦 → NPC 決定接單）
    ↓
等待（實時計時）
    ↓
任務結算（成敗 + 死亡判定）
    ↓
收取傭金 / 承擔賠償
    ↓
（回到頂部）
```

### 遊戲特色

- **離線與掛機**：用零散時間上來審核委託，然後把委託丟給冒險者去做
- **NPC自主性**：冒險者有個性→他們會拒絕、會自己接受任務（目前沒有決策系統）
- **真實後果**：公會倒閉、金幣有限、聲望影響
- 招募公會職員：每位公會職員都有不同的專長，招募你心目中的人才，讓他們成為你的工作伙伴吧
- **存活或死亡**：每一次的委託可能都是一場生死冒險，在你上班上課前接了委託的冒險者，當下班下課再打開遊戲，可能他就再也回不來了
